## <a name="about-altspace"></a>Acerca de Altspace

[AltspaceVR](https://altvr.com/) es el punto de encuentro para asistir a programas en vivo, encuentros, clases interesantes y mucho más con personas amistosas de todo el mundo. Todo gracias a la magia de la realidad virtual.  Altspace se incorporó a la familia de Microsoft en 2017 y nos complace mostrar cómo se pueden celebrar eventos en un espacio virtual.

Altspace admite las plataformas siguientes: HTC Vive; Oculus Rift, Go, Quest, Samsung GearVR, Windows Mixed Reality y el modo en 2D del equipo.

### <a name="creating-an-account"></a>Creación de una cuenta

Hay dos formas de crear una cuenta:

La manera más eficaz es descargar el cliente de AltspaceVR que prefieras para la configuración, completar la instalación y, a continuación, iniciar el cliente. Una vez que esté en el programa, habrá una opción para crear una cuenta.  Se pueden encontrar las especificaciones mínimas del sistema [aquí](https://help.altvr.com/hc/en-us/articles/115003538414-Minimum-System-Specifications).

Otra opción es visitar el [registro de cuentas en línea](https://account.altvr.com/users/sign_up).

### <a name="troubleshooting"></a>Solucionar problemas

Si tienes problemas con Altspace, consulta la [página de solución de problemas](https://help.altvr.com/hc/en-us/sections/115000792634-Troubleshooting). Si no se soluciona tu problema, envía una [solicitud de soporte técnico](https://help.altvr.com/hc/en-us/requests/new).

## <a name="code-of-conduct-and-microsoft-standard-event-guidelines"></a>Código de conducta y directrices de eventos estándar de Microsoft

### <a name="code-of-conduct"></a>Código de conducta 

***La misión de Microsoft es permitir que cada persona y cada organización del planeta pueda lograr más. Esto incluye todos los eventos y las reuniones en Microsoft, incluidas las plataformas digitales, en que intentamos crear una experiencia respetuosa, amistosa, divertida e inclusiva para todos los participantes.***  

***Esperamos que todos los participantes de eventos digitales respeten los principios de este código de conducta, que abarca el evento digital principal y todas las actividades relacionadas. No se tolera ningún comportamiento, mensaje, imagen o interacción disruptiva ni irrespetuosa de ningún participante, de ninguna forma y en ningún aspecto del programa, incluidas las actividades empresariales y sociales, independientemente de su ubicación. Microsoft no tolerará ningún acoso ni discriminación basado en edad, ascendencia, color, expresión o identidad de generó, origen nacional, discapacidad física o mental, religión, orientación sexual o cualquier otra característica protegida por las normas, regulaciones y ordenanzas aplicables.***  

***Está prohibido hacer fotografías o vídeos o grabar de cualquier forma el contenido o las presentaciones de sesión que se comparten en los canales sociales. Animamos a todos los usuarios a facilitar la creación de un entorno seguro y cordial. Notifica cualquier problema, comportamiento de acoso, actividad sospechosa o disruptiva mediante la línea directa de conducta empresarial: 1-877-320-MSFT o [buscond@microsoft.com](mailto:buscond@microsoft.com). Microsoft se reserva el derecho de rechazar la admisión o bloquear a cualquier persona de Mixed Reality Dev Days en cualquier momento a su entera discreción.***

## <a name="frequently-asked-questions"></a>Preguntas más frecuentes
Esta sección se actualizará con frecuencia a medida que recibamos más preguntas.

**P**. ¿Puedo participar sin un casco de VR?  
**R**. Desde luego. Aunque creemos que el evento se disfruta mejor con un casco de VR, queremos incluir a los usuarios que no pueden unirse a nosotros en ese formato.  Altspace tiene un cliente para equipos que se puede usar para unirse al evento. ***Actualmente, Altspace no tiene ningún cliente que funcione con la plataforma Mac***.  
  
**P**. ¿Cómo puedo unirme a la experiencia en 2D?  
**R**. Desde un equipo, tienes la opción de [descargar el cliente de Altspace](https://altvr.com/get-altspacevr/) desde Microsoft Store o Steam. Después de descargar el cliente, tendrás que crear una cuenta e iniciar sesión. Puedes encontrar información sobre cómo navegar dentro de la experiencia 2D [aquí](https://help.altvr.com/hc/en-us/articles/115003528373-How-do-I-control-my-avatar-with-Mouse-Keyboard-).
  
**P**. ¿Quién debería asistir a Mixed Reality Dev Days?  
**R**. Este evento incluye a todos los usuarios que trabajen o jueguen en un espacio de realidad mixta. Aunque la mayor parte del contenido se centrará en el desarrollador, también sabemos que hay contenido interesante para una amplia variedad de roles más allá de los desarrolladores. Recuerda comprobar las actualizaciones de la programación para estar al corriente del tipo de contenido que se va a proporcionar.
